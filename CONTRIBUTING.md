Contribution Guide
============

### How to contribute to the project:

1. Make a copy of the project TodoList

2. Follow the [README](./README.md) of this project for the installation

3. Create a new branch

```bash
$ git checkout -b feature-YourName-NameOfFunctionnality
``` 

4. Push this new branch in this repository GitHub
```bash
git push origin your-branch
``` 
        
5. Create a new pull request by comparing your branch and DEV branch


### Quality process :

The code have to be W3C valide (https://validator.w3.org/)

During the development you will ensure to respect the good coding practices of PSR-1 and 12, if you use PHP.

Once your improvements have been made, you can make a pull request on the main repository of the project.

In your pull request you will be sure to remember if this is a feature or a bug fix and you will choose the right label accordingly.
